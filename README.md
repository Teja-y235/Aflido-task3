# Task Overview

This repository contains my implementation for **Task 3 – File Handling in Java**. The objective of this task is to understand and apply file operations using Java's IO (Input/Output) classes.

The project demonstrates how to create a file, write data into it, and read the stored data back from the file. File handling is an important concept in Java that enables applications to store and retrieve information permanently.

This task helped me gain practical experience with Java file operations, exception handling, and stream-based input/output processing.

# Project Description

The File Handling program performs the following operations:

* Creates a text file
* Writes data into the file
* Reads data from the file
* Displays file content on the console
* Handles file-related exceptions

The program uses Java IO classes such as FileWriter, FileReader, and BufferedReader to perform file operations efficiently.

---

# Concepts Used

### 1. File Writing

The `FileWriter` class is used to create a file and write text data into it.

```java id="wr1"
FileWriter writer = new FileWriter(fileName);
```

### 2. File Reading

The `FileReader` class is used to read data from the file.

```java id="rd1"
FileReader reader = new FileReader(fileName);
```

### 3. BufferedReader

The `BufferedReader` class improves reading efficiency by reading text line by line.

```java id="bf1"
BufferedReader br = new BufferedReader(reader);
```

### 4. Exception Handling

The program uses try-catch blocks to handle file-related errors and prevent program crashes.

```java id="ex1"
catch (IOException e)
```

### 5. Input and Output Streams

Java IO streams are used to transfer data between the program and the file system.
